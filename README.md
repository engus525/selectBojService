# selectBojService
코테 주요 유형에 속한 적정 난이도의 문제를 골라 풀고싶으나,<br>
직접 solved.ac에서 필터링 할 경우 알고리즘 유형과 난이도를 알고있는 상태에서 시작해야하므로<br>
백준 핸들만 입력하면 적절한 유형에서 적절한 난이도의 문제 리스트를 뽑아주는 서비스이다.
